Stabilimenti e Basi dati

Ogni stabilimento ha la propria base dati githubDB (file data/*.json). Puoi usare lo stesso repo con file diversi o fork separati per ogni cliente.

Multi-tenant
  • Stesso repo: winbeach-lido1.json, winbeach-lido2.json nello stesso fork
  • Repo separati: ogni cliente con il proprio fork githubDB
  • Token: uno per stabilimento (salvato nel browser)
